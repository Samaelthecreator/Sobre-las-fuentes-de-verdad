# Sobre las fuentes de verdad

> "La verdad es el error que escapa del engaño."

## Intenciones del Proyecto
Este proyecto se posiciona como formas de confrontar el problema de la desinformación y la manipulación digital. 

Vivimos en una era donde la verdad ha sido mercantilizada y diluida por algoritmos diseñados para la retención, no para la educación. Medios digitales, influencers y hasta instituciones formales bombardean con datos sesgados, descontextualizados o abiertamente falsos para moldear la opinión pública a conveniencia.

**La intención de este proyecto es:**
1.  **Cuestionamiento y espacio a la crítica de la información:** No dar nada por sentado. Cada tweet, cada noticia, cada gráfico debe ser cuestionado.
2.  **Validar la Información:** Establecer un estándar epistemológico para filtrar la basura digital. Si no es verificable, entonces se determina su intención y objetivo de existir.
3.  **Base Sólida para el Análisis:** Construir cimientos de datos reales para análisis sociopolíticos. No podemos entender ni cambiar la sociedad si partimos de falsedades e incongruencias.

Aquí no buscamos la neutralidad, buscamos la **objetividad**. Queremos herramientas para desmantelar narrativas falsas y entender la realidad tal como es, no como nos la quieren vender.
