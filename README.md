# Sobre las fuentes de verdad

> "En tiempos de engaño universal, decir la verdad es un acto revolucionario."

## Intenciones del Proyecto (Crudo y Directo)
Este proyecto no es solo un repositorio de definiciones; es una declaración de guerra contra la desinformación y la manipulación digital. 

Vivimos en una era donde la verdad ha sido mercantilizada y diluida por algoritmos diseñados para la retención, no para la educación. Medios digitales, influencers y hasta instituciones formales bombardean con datos sesgados, descontextualizados o abiertamente falsos para moldear la opinión pública a conveniencia.

**La intención de este proyecto es brutalmente simple:**
1.  **Cuestionar TODO:** No dar nada por sentado. Cada tweet, cada noticia, cada gráfico debe ser sometido a un escrutinio riguroso.
2.  **Validar la Información:** Establecer un estándar epistemológico para filtrar la basura digital. Si no es verificable, no sirve.
3.  **Base Sólida para el Análisis:** Construir cimientos de datos reales para análisis sociopolíticos serios. No podemos entender ni cambiar la sociedad si partimos de mentiras.

Aquí no buscamos la neutralidad tibia, buscamos la **objetividad radical**. Queremos herramientas para desmantelar narrativas falsas y entender la realidad tal como es, no como nos la quieren vender.
